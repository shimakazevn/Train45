## 2. Reina's Relationship Arc

### Korean Address System for Reina

The shift in her tone is **not** in grammar (she stays formal) but in **content**: from professional distance to emotional vulnerability to erotic confession.

| Aspect | Rule | Example |

| Aspect | Rule | Example |
|--------|------|---------|
| Address MC | `anh` (friendly, not formal) | Can drop to just `anh` |
| Self-reference | `tôi` or `mình` (casual) | `tôi` is fine, not overly formal |
| Sentence endings | Casual finals: `nhé`, `thế`, `à` | No `ạ`, no formal particles |
| Emotional keywords | `thích`, `nhớ`, `muốn`, `tuyệt` | More direct, energetic vocabulary |
| Intimate scenes | Casual and open | She's shameless and direct |
| Key phrase | `근데 너하고 하는건 좋아!` → `Nhưng được làm với anh thì tôi thích!` | Emphasis on "with YOU specifically" |

### Tone Progression Arc Summary

**Reina**: Stranger (formal) → Acquaintance (formal but warm) → Partner (formal, erotic, vulnerable) → Lover (formal, deeply affectionate)

**Mai**: Stranger (casual) → Friend (casual, flirty) → Lover (casual, open, honest) → Soulmate (casual, emotionally raw)

### Critical Don'ts
- ❌ Never make Reina drop to casual speech with MC
- ❌ Never make Mai use formal speech with MC
- ❌ Never use Vietnamese regional slang that breaks character (e.g., Southern `bây` for Mai)
- ❌ Never add intimacy markers (pet names, `ưng` etc.) that aren't in the original
- ❌ Never translate `당신` as the same word each time — vary by context (`anh`, `người đó`, `ngài`)
